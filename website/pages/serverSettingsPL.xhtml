Java VMOptions are passed to the VM of most Java language modules.
You can use these options to increase the amount of memory allotted to a
Java language module if you are experiencing OutOfMemory errors.

For details of java options run
    java -X
at a command line.
 

R Options are passed to the R interpreter of most R language modules.
You can use these options to increase the amount of memory allotted to an
R language module if you are experiencing vector allocation errors.

For details of R options run
    R --help
at a command line.
