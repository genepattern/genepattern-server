Or Install Python Library From PIP or Conda

PIP is the official Python package manager. It comes with many distributions of Python. To learn more about PIP click here.

To install the GenePattern Python library from PIP, simply enter the line shown below into your command line. Installing packages from PIP in this way may require administrator privilages (ex: sudo).

pip install genepattern-python

Conda is an open source package management system and environment management system that runs on Windows, macOS and Linux. To learn more about Conda click here

To install the GenePattern Python library from Conda, simply enter the line shown below into your command line.

conda install -c genepattern genepattern-python
