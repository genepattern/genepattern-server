Download Sample Datasets

To get started using GenePattern right away, you can download sample datasets from the public GenePattern website.

Step 1. Check Requirements

The sample datasets are in gzip format. In order to install these, you must have a compression utility such as WinZip. If you do not have one, you can download one of the following applications:
  • Windows: Download an evaluation version of WinZip from http://www.winzip.com/downwz.htm
  • Mac: Download StuffIt Expander from http://www.stuffit.com/cgi-bin/stuffit_loginpage.cgi?standardmacexp
  • Linux: Your Linux distribution should already have the gzip and gunzip tools. To check this, type which gunzip at the Unix prompt.

Step 2. Download Datasets

We recommend that you create a folder named Datasets and expand the sample dataset directories within that folder. The following datasets are used in the GenePattern tutorial. To download a dataset, click on the Datasets link for each of the following papers.

These datasets are also available on the public GenePattern website at https://software.broadinstitute.org/cancer/software/genepattern/datasets. A link to this page is provided in the Documentation section on your GenePattern homepage for later downloads.
